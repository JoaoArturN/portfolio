Sobre o Projeto:

O Projeto Rede Social é um projeto que fiz para interagir com meus amigos enquanto estudo. Personalizei a Rede Social com nosso estilo, com histórias e fotos pessoais, é um projeto pessoal portando seu conteúdo não é tecnico mas se encontra publicado públicamente. Além disso essa Rede Social funciona perfeitamente, Cada membro pode possuir seu perfil e personalizar da maneira que preferir e realizar posts com imagens e textos, assim é ótimo para armazenar momentos pessoais.

Linguagem:

O projeto foi feito com PHP, com utilização de Handlers, Controllers, e Programação Orientada a Objetos. Também foi utilizado JavaScript, principalmente para realizar Fetchs. Além disso o banco de dados MYSQL que armazena todos os posts e usuarios cadastrados.
